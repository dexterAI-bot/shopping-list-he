## Scope v1
- קבוצה אחת בטלגרם (chat_id: `-5284617579`)
- הוספת פריטים בטקסט או תמונה (OCR עם אישור תמיד)
- פקודה: **"התחל קניות"** מחזירה לינק לעמוד מצב קניות
- מצב קניות: כפתור **הכנס לעגלה / הוצא מהעגלה** לכל פריט
- אפשר להזין מחיר/כמות שנקנתה/הערה בעת הכנסה לעגלה
- סיום קנייה: רק פריטים שבעגלה יורדים מהרשימה, השאר נשארים
- שמירת היסטוריית מחירים + חנות/סניף
- הכל בעברית, RTL, mobile-first
